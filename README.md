# Next.js TRI — Projeto

Projeto mínimo em Next.js (App Router) com TailwindCSS, paleta fria e páginas simples.

## Como rodar localmente

1. `npm install`
2. `npm run dev`
3. Abra `http://localhost:3000`

## Deploy no Vercel

- Suba este repositório para o Git (GitHub/GitLab) e conecte no Vercel.
- Vercel detecta automaticamente o Next.js.
